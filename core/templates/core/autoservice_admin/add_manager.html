Описание ролей
Администратор автосервиса
  • Полный доступ к управлению автосервисом
  • Может редактировать информацию об автосервисе
  • Может добавлять и удалять менеджеров
  • Доступ ко всем функциям панели управления
Менеджер
  • Доступ к основным функциям управления
  • Может работать с записями клиентов
  • Может просматривать информацию об автосервисе
  • Не может изменять настройки автосервиса
Важная информация
  • Пользователь должен быть зарегистрирован в системе с указанным email
  • Проверьте правильность email перед добавлением
  • Один пользователь может быть менеджером только одного автосервиса
  • После добавления пользователь получит доступ к панели управления
  • Вы можете изменить роль менеджера в любое время
